# Add task ID search to task list search bar

## Problem

   ```tsx
   if (
     filters.searchTerm &&
     !task.title.toLowerCase().includes(filters.searchTerm.toLowerCase()) &&
     !task.id.toLowerCase().includes(filters.searchTerm.toLowerCase())
   )
     return false;
   ```
